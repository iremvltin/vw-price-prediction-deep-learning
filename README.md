# Volkswagen Used Car Price Prediction

[English](#english) | [Türkçe](#türkçe)

<a id="ingilizce"></a>
## English

### Project Purpose:
This project was developed to analyze the key parameters affecting the prices of Volkswagen vehicles in the UK used car market and to generate highly accurate price predictions using a Deep Learning approach. 

### Model Information:
* **Architecture:** Deep Neural Network (DNN) consisting of 7 hidden layers and 1 output layer (8 layers in total).
* **Performance:** 93.19% success rate (MAPE: 6.81%).

### Repository Contents:
* `english-vwcarprices.ipynb`: The complete project notebook in English, including exploratory data analysis, preprocessing, and model evaluation.
* `turkce_vwcarprices.ipynb`: The Turkish version of the project notebook.

## Türkçe

### Proje Amacı:
Bu proje, İngiltere ikinci el araç pazarındaki Volkswagen marka araçların fiyatlarını etkileyen temel parametreleri analiz etmek ve Derin Öğrenme yaklaşımı kullanarak yüksek doğrulukta fiyat tahminleri üretmek amacıyla geliştirilmiştir.

### Model Bilgisi:
* **Mimari:** 7 gizli katman ve 1 çıkış katmanından oluşan toplam 8 katmanlı Derin Sinir Ağı (DNN).
* **Performans:** %93.19 başarı oranı (MAPE: %6.81).
